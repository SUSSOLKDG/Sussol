README SUSSOL FREEMIUM
----------------------

Installatie
***********

Unzip de zip file 'Sussol Freemium.zip' ergens op je harde schijf, bvb op je bureaublad. Hierdoor wordt er een nieuwe map gemaakt, met daarin 2 bestanden:

Sussol Freemium.jar
freemium.csv


Als Java geÃ¯nstalleerd is (wat op de meeste machines het geval is), dan zou het programma (Sussol Freemium.jar) moeten opstarten door er op te dubbelklikken.
Anders moet je een command line openen, daarin naar de juiste map navigeren en dan het volgende commando ingeven:

java -jar "Sussol Freemium.jar" .

Gebruik
*******

1. Kies een solvent uit de lijst van het openingsscherm, en klik op de Cluster knop. De analyse wordt gestart, zie progressbar.
2. In het menu 'Visualisation' kies je 'Clustering'. Je komt op een scherm waar links de clusters staan, en rechts de solventen van de geselecteerde cluster.
3. In het menu 'Candidates' kan je via 'Generate Candidates' naar het scherm gaan waar de lijst van groene alternatieven, met hun parameters in staan.

Klik op een solvent in de lijst, en het wordt gehighlight in de line chart.
Rechtermuisklik toont enkel het geselecteerde solvent.

4. Ook zonder clustering kan je de dataset bekijken, via 'Visualisation -> All Solvents'. Om de dataset te filteren op de parameters van de solventen, klik je in dit scherm op de knop 'Filter Solvents'. Voor elke parameter verschijnt er een slider die je als filter kan toepassen. Als je dit extra scherm afsluit, wordt de filtering gewist en staat de volledige dataset terug beschikbaar.
